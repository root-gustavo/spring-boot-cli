# Spring Boot CLI

Este projeto é um **CLI experimental** criado para estudar e automatizar tarefas comuns no desenvolvimento com **Spring Boot**.

A ideia inicial é fornecer comandos interativos para:
- gerar estruturas básicas de projetos
- criar arquivos padrão (packages, classes, configs)
- acelerar tarefas repetitivas do dia a dia

O projeto começa simples e será evoluído gradualmente conforme novas ideias e necessidades surgirem.
